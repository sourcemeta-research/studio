# Sourcemeta Studio Extension

Visual Studio Code extension for JSON Schema inspection, formatting, and linting.

## Running Locally

1. Open this directory in VS Code
2. Install all the dependencies using `npm run install:all` and build the extension using `npm run build`.
3. Go to Run and Debug (Cmd+Shift+D) and select "Run Extension"
4. Press F5 or click the green play button
5. Click the Sourcemeta Studio icon in the activity bar (left sidebar)
